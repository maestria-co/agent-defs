## How to Add an ADR

1. Create a new file: `ADR-NNN-short-title.md` (e.g., `ADR-002-use-postgres.md`)
2. Copy the template from any existing ADR file
3. Add a row to the index above
4. Status values: `Proposed` → `Accepted` → `Deprecated` / `Superseded by ADR-NNN`
